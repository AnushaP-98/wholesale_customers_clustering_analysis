# Wholesale Customers Clustering Analysis

## Project Overview
This project applies unsupervised learning techniques to the Wholesale Customers dataset to identify natural customer segments based on purchasing behavior. The objective was to determine optimal customer groupings using clustering techniques and evaluate the quality of the solution.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn
- Scikit-learn
- K-Means Clustering
- PCA (Dimensionality Reduction)

## Part 1: Data Exploration & Preparation
- Descriptive statistics and distribution analysis
- Correlation analysis
- Outlier detection and treatment
- Feature scaling using StandardScaler

## Part 2: Clustering & Validation
- K-Means clustering applied
- Elbow method used to determine optimal clusters
- Silhouette score used for validation
- PCA applied for 2D visualization of clusters

## Key Findings
- Customers were segmented into distinct purchasing behavior groups.
- Certain clusters showed high spending on Fresh products, while others focused on Grocery and Milk.
- Dimensionality reduction improved cluster visualization.

## Business Insights
- Identified potential high-value customer groups.
- Provided segmentation that can support targeted marketing strategies.
- Highlighted spending patterns across product categories.
